# Indica Carapicuíba

Projeto desenvolvido para a atividade A2 - Design com Stitch.

O objetivo do projeto é criar uma plataforma digital de indicações de comércios, serviços e profissionais locais da cidade de Carapicuíba.

O desenvolvimento do projeto utilizou:
- Stitch Google para criação das telas e prototipação visual
- ChatGPT para auxílio na geração e estruturação do código
- GitHub para versionamento e documentação
- Vercel para deploy e publicação do site
- 
- ## Storytelling da Atividade

O projeto “Indica Carapicuíba” foi desenvolvido com a proposta de criar uma plataforma digital voltada à divulgação de comércios, serviços e profissionais locais da cidade de Carapicuíba.

A primeira etapa do processo foi realizada utilizando a plataforma Stitch Google, onde foram desenvolvidas as telas iniciais do projeto com foco em organização visual, navegação simples e identidade moderna.

Após a criação das telas, o projeto foi submetido à IA para conversão da proposta visual em estrutura HTML e CSS, realizando também ajustes de navegação, organização de arquivos e estrutura com ponto de entrada no arquivo index.html.

O projeto foi posteriormente enviado ao GitHub para versionamento e compartilhamento acadêmico, e publicado na plataforma Vercel para disponibilização online.
